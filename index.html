MONITOR·MATCH

Hold the camera flush against each screen showing the same test color. The app reads what each display actually emits and tells you how to adjust the older one.

1 · Display the test color full-screen

Put this color full-screen on the monitor you're about to read. Use the SAME color on both monitors. White exposes overall tint; gray is best for matching brightness/contrast.

(If you're reading a desktop monitor, just set that monitor's screen to the chosen color — e.g. open a blank colored image. This button is only if you're matching two phones/tablets.)

2 · Read each monitor

Press the camera flat against the screen center. The app averages a large patch, so focus and moiré don't matter. Hold steady and capture.
